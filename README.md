# Attendance Kiosk - Mobile App

Face Recognition Attendance System with Fingerprint, NFC, and Barcode support.

## Features

- ✅ Fingerprint Authentication
- ✅ NFC Card Reading
- ✅ Barcode Scanning
- ✅ Offline Sync with Queue Management
- ✅ Analytics Dashboard
- ✅ Performance Optimization
- ✅ Toast Notifications
- ✅ Haptic Feedback
- ✅ Network Status Indicator

## Tech Stack

- React Native 0.72.6
- AsyncStorage for local data
- NetInfo for network detection
- Custom Analytics Service
- Custom Offline Sync Manager

## Installation

### Prerequisites

- Node.js 16+
- Java JDK 11+
- Android SDK

### Setup

```bash
npm install
cd android
./gradlew assembleDebug
```

## Download APK

Go to [Actions](../../actions) tab → Click latest workflow run → Download APK from "Artifacts"
